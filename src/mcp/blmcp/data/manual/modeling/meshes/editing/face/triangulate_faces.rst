*****************
Triangulate Faces
*****************

.. reference::

   :Mode:      Edit Mode
   :Menu:      :menuselection:`Face --> Triangulate Faces`
   :Shortcut:  :kbd:`Ctrl-T`

Converts each selected face (whether it's a quad or an :term:`n-gon`) to triangular faces.
See the :doc:`/modeling/modifiers/generate/triangulate`.
