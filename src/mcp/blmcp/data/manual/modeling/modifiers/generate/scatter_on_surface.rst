***************************
Scatter on Surface Modifier
***************************

The *Scatter on Surface* modifier distributes points or instances across the surface of a mesh.
It provides flexible control over how elements are placed using density, randomization,
and Poisson disk sampling.
This modifier is useful for procedural scattering of objects such as grass, rocks, or debris
on terrain or any other surface.
