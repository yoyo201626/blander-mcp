.. _bpy.types.SpaceDopeSheetOverlay:

*******************
Dope Sheet Overlays
*******************

.. reference::

   :Mode:      All Modes
   :Location:  :menuselection:`Header --> Overlays`

Clicking :bl-icon:`overlay` (Show Overlays) toggles all overlays in the Dope Sheet.

The drop-down button displays a popover with more detailed settings, which are described below.

.. _bpy.types.SpaceDopeSheetOverlay.show_scene_strip_range:

Show Scene Strip Range
   When using :ref:`scene time synchronization <bpy.types.WorkSpace.use_scene_time_sync>`
   in the :doc:`Sequence Editor </editors/video_sequencer/index>`, display the range of the current scene strip.
