****************
Combine XYZ Node
****************

.. figure:: /images/node-types_ShaderNodeCombineXYZ.webp
   :align: right
   :alt: Combine XYZ Node.

The *Combine XYZ Node* combines a vector from its individual components.


Inputs
======

- X
- Y
- Z


Output
======

Vector
   Standard vector output.

.. note::

   The vector is not normalized.
