
****
Blur
****

.. reference::

   :Mode:      Sculpt Mode
   :Brush:     :menuselection:`Sidebar --> Tool --> Brush Settings --> Advanced --> Brush Type`

The *Blur* brush smooths painted colors on the active
:ref:`Color Attribute <modeling-meshes-properties-object_data-color-attributes>`.
It samples the colors under the cursor and blends them together,
softening transitions and reducing sharp color differences.

This brush is commonly used to refine painted details,
smooth gradients, and remove harsh edges in vertex color painting workflows.
