.. _bpy.ops.transform.vert_crease:

*************
Vertex Crease
*************

.. reference::

   :Mode:      Edit Mode
   :Menu:      :menuselection:`Vertex --> Vertex Crease`

Adjusts the Crease value of the selected vertices, which determines the sharpness of their
corner when using the :doc:`/modeling/modifiers/generate/subdivision_surface`.

The Crease value can also be changed in the :ref:`modeling-mesh-transform-panel`
of the Sidebar.
